== Description ==
Rigorous Instagram and Pinterest feeds is a simple plugin that add Instagram feeds and Pinterest feeds in your site.It is simple with responsive design that shows Instagram image feeds and Pinterest feeds.

== Installation ==
1.Download  Rigorous Instagram and Pinterest Feed from WordPress.
2.Unzip rigorous-instagram-and-pinterest-widget.zip
3.Upload all the files to the /wp-content/plugins/
4.Activate the plugin through the ‘Plugins’menu in WordPress

== Frequently Asked Questions ==
= What does this plugin do? =
This plugin provides the ability to add instagram feed and pinterest feed from widget section.

= Do I need to design the layout again? =
No , you won't need to get any trouble regarding design and layout since we have provided inbuilt design for sidebar and footer section.You can choose any as per your requirement.
